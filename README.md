## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/shahbazhassan42000/love.git
   ```
2. Navigate to the project directory:

   ```bash
   cd love
   ```
3. Install the required dependencies:

   ```bash
   npm install
   ```
4. Start the development server:

   ```bash
   npm run dev
   ```
